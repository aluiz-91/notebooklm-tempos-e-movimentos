# Tempos e Movimentos na Administração Científica de Frederick Taylor

## Contexto

Este projeto foi desenvolvido utilizando o NotebookLM para estudar os conceitos de Tempos e Movimentos propostos por Frederick Winslow Taylor, um dos principais representantes da Administração Científica.

## Objetivos

- Compreender os princípios da Administração Científica.
- Estudar os conceitos de tempos e movimentos.
- Identificar vantagens e limitações da teoria de Taylor.
- Aplicar técnicas de engenharia de prompts para aprofundar o aprendizado.

## Fontes Utilizadas

As fontes utilizadas encontram-se na pasta `/fontes`.

## Engenharia de Prompts

Os testes de prompts, resultados e dificuldades encontradas encontram-se na pasta `/prompts`.

## Lições Aprendidas

Durante a elaboração deste projeto foi possível compreender a importância dos estudos de tempos e movimentos para o desenvolvimento da Administração Científica.

A pesquisa demonstrou como Frederick Taylor buscava aumentar a eficiência das organizações por meio da análise sistemática das tarefas executadas pelos trabalhadores.

Além disso, o uso do NotebookLM permitiu explorar diferentes formas de formular perguntas, mostrando que a qualidade das respostas depende diretamente da clareza dos prompts utilizados e das fontes consultadas.
